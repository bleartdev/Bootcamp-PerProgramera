CSS Attribute Selectors — Examples

1) [attr] — Presence selector

Ky paragraf ka title

Ky paragraf s’ka title

2) [attr="value"] — Exact match

3) [attr^="value"] — Prefix (fillon me)

Link me https:// (zgjedhet)
Link me http:// (nuk zgjedhet)

4) [attr$="value"] — Suffix (mbaron me)

cv.pdf (zgjedhet)
photo.jpg (nuk zgjedhet)

5) [attr*="value"] — Substring (e permban)

tutorial-css.html (zgjedhet)
index.html (nuk zgjedhet)

6) [attr~="value"] — Word match (fjale ne liste)

class="card featured" (zgjedhet)
class="cardfeature" (nuk zgjedhet)

7) [attr|="value"] — Language match

lang="en" (zgjedhet)

lang="en-US" (zgjedhet)

lang="eng" (nuk zgjedhet)
